1.8

[meta]

-- wherever remove_username_space is available, a new flag called username_asciify was added. it removes all non-alphanumerical characters instead of only removing spaces.

-- added a YouTube XL frontend from 2009 for testing. available at /xl, working only with Flash for now.

-- at /toggle_f, you can now also set a custom swf file as a player.

[wayback_features]

-- descriptions for some videos aren't lowered 1 line anymore (Thanks, Griffen!)

-- if available, an old user banner will be shown. otherwise a wayback_not_found classname will be added to the container, so - for example - they can be filtered out with CSS

-- on yt2009_flags, you can set an all parameter to use all Wayback Features. (Thanks, Griffen!)

-- the comment count properly counts comments added through wayback_features (Thanks, Griffen!)

-- fixed adding to favorites and playlists.

[watchpage]

-- [feather] added support for wayback_features

-- fallback date formats (yyyy-mm-dd) will show normally now (ex. 2009-10-10 will be shown as Oct 10, 2009)

-- clicking Video Responses will bring up some video responses from the time. using wayback_features is recommended for better results.

-- updated the word list used by comments_remove_future to filter comments.

-- fixed a bug where the watchpage would show a random rating number if use_ryd doesn't report any ratings.

-- with f enabled, videos added using exp_related will show on the endscreen. (Thanks, HazelNut!)

-- fixed share_behavior and login_simulate (Thanks, breakgimme!)

[player]

-- higher quality stars on the endscreen

-- if the videos shown on the endscreen have a cached use_ryd rating, it will be shown instead of the default 5 stars.

-- fixed the blinking speech tip in annotations (Thanks, Griffen and breakgimme!)

-- fixed the muted volume button being 1px off.

-- older browsers will now show an annotations gradient (Thanks, BoredWithADHD!)

[playlists]

-- clicking on a title or a thumbnail will redirect to that video.

[search]

-- added pages (Thanks, BoredWithADHD!)

-- only_old is more reliable (Thanks, Griffen!)

[channels]

-- better flag and edge case handling.
