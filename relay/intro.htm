yt2009_relay

relay is a piece of node code that you can run when using yt2009 as normal.
it's not necessary and your current functionality won't be impacted without it, but it can add a few features you might find useful. such as:

  • copying over your youtube data:
    • playlists
    • subscriptions
  • interacting with videos as you (favoriting, commenting)

and more in the future
